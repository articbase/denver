# Platform Engineering by Sumesh

A premium, modern website showcasing Platform Engineering services.
Built with Vite, Vanilla JS, and CSS.

## Features

- **Modern Design**: Dark mode, glassmorphism, and neon accents.
- **Interactive Elements**: Scroll reveal animations, parallax effects, and typing code blocks.
- **Responsive**: Fully responsive layout for all devices.
- **High Performance**: Optimized with Vite.

## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Customization

- **Content**: Update `index.html` with your specific text.
- **Styles**: Modify `src/style.css` to change colors or layout.
- **Scripts**: Edit `src/main.js` for additional interactivity.

## Deployment

Set up with Netlify, Vercel, or any static site host easily by connecting your repo and setting the build command to `npm run build` and publish directory to `dist`.
